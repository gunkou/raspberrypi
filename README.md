# raspberrypi

[ドキュメント](https://www.raspberrypi.org/documentation/)

## インストール

[Raspberry Pi Downloads – Software for the Raspberry Pi](https://www.raspberrypi.org/downloads/)


## インストール後にやること

ラズベリーパイの「ファームウェア」を最新版

```
sudo rpi-update
```

再起動

```
sudo reboot
```

パッケージ管理ツールを最新版

```
sudo apt-get update
```

アップグレード

```
sudo apt-get upgrade
```

vimのインストール

```
sudo apt-get install vim
```
